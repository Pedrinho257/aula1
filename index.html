Linguagens importantes para o desenvolvimento web
HTML
    
O HTML é uma linguagem de marcação, que serve de base para a criação de sites. 
     Com o HTML, é possível a criação, exibição, formatação e editação de elementos de uma página de internet, como:
     
     Textos
     Imgens
     Formulários
     Tabelas

JavaScript
     
 A linguagem de programação de JavaScript é usada para rodar scripts através dos navegadores.
        Com o JavaScript, algumas aplicações podem ser rodadas sem a necessidade de haver troca de informações com o servidor.
        Assim, um documento poderia ser alterado através destes scripts qu8e interagem diretamente com o usuário.
        Iremos aprender JavaScript logo após estudar HTML.   

     
     
 A linguagem de programação PHP é do tipo interpretada e é, atualmente, uma das mais usadas no mundo.
        O PHP é vastamente usado para criar aplicações e sistemas, do lado do servidor.
        Usa-se muito o PHP para criar páginas com conteúdo de uma maneira dinâmica, além de ser possivel acessar 
        e interagir com servidores e banco de dados.
